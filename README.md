# 🖤 Neovim Monochrome

Тёплая монохромная конфигурация Neovim, ориентированная на комфорт для глаз и эргономику. Без ярких цветов, только мягкие оттенки и всё необходимое для продуктивной работы.

## 🚀 Установка

1. **Клонируйте репозиторий** в папку `~/.config/nvim`:
   ```bash
   git clone https://github.com/your/repo.git ~/.config/nvim
   ```
2. **Откройте Neovim** — менеджер плагинов `lazy.nvim` установится автоматически.
3. **Установите плагины** командой:
   ```
   :Lazy sync
   ```
4. **Установите LSP‑серверы и форматтеры** (через Mason):
   ```
   :MasonInstall stylua prettierd isort black rustfmt
   ```
   > Примечание: базовые LSP (lua_ls, pyright, ts_ls, rust_analyzer) устанавливаются автоматически.

## ✨ Основные возможности

- **Тема Zenwritten** — мягкая светлая тема без лишних акцентов.
- **Telescope** — быстрый поиск файлов, текста и буферов.
- **Hop** — мгновенное перемещение по видимой области экрана двумя символами (`s`).
- **Conform** — автоформатирование при сохранении (stylua, prettierd, black, rustfmt).
- **Blink.cmp** — автодополнение с поддержкой сниппетов (LuaSnip + Friendly Snippets).
- **Zen Mode** — режим концентрации (`<leader>z`).
- **Mini.Statusline** — минималистичная строка состояния.
- **LSP** — встроенная поддержка Lua, Python, TypeScript и Rust.
- **Gitsigns** — значки изменений в коде (Git).
- **Which-Key** — подсказки по горячим клавишам.
- **Oil** — файловый менеджер-буфер (`<leader>e`).
- **NeoScroll** — плавная прокрутка.
- **Colorizer** — подсветка цветов (#RRGGBB).

## ⌨️ Основные клавиши

| Клавиша          | Действие                               |
| ---------------- | -------------------------------------- |
| `<leader>ff`     | Поиск файлов                           |
| `<leader>fg`     | Поиск текста по проекту                |
| `<leader><leader>` | Список открытых буферов              |
| `s`              | Прыжок в любое место на экране (Hop)   |
| `<leader>z`      | Вкл/выкл Zen Mode                      |
| `<leader>e`      | Открыть файловый менеджер (Oil)        |
| `gd`             | Перейти к определению                  |
| `gr`             | Найти все упоминания                   |
| `K`              | Показать документацию                  |
| `<leader>rn`     | Переименовать переменную               |
| `<leader>ca`     | Действия с кодом / исправления         |
| `[d` / `]d`      | Переход по диагностическим ошибкам     |
| `<leader>e`      | Показать ошибку под курсором           |
| `<C-h/j/k/l>`    | Перемещение между окнами               |

## ⚙️ Настройка под себя

- **Форматтеры**: редактируйте список `formatters_by_ft` в блоке `conform.nvim`.
- **LSP‑серверы**: меняйте `ensure_installed` в `mason-lspconfig`.
- **Цвета темы**: корректируйте `hl(...)` в блоке `zenbones.nvim`.
- **Статуслайн**: по умолчанию используется минимальный стандартный вид. При желании можно расширить его в секции `mini.statusline`.
- **Сниппеты**: добавлены `friendly-snippets` для популярных языков; дополнительные коллекции ставятся через Mason или вручную.

## 📦 Зависимости

- Neovim ≥ 0.9
- Git
- Опционально: `stylua`, `prettierd`, `isort`, `black`, `rustfmt` (устанавливаются через `:Mason`)
- Рекомендуется шрифт **Nerd Font** для корректного отображения иконок
